Багровый комар — отвратительное летающее насекомое-мутант, обитающее в Багровых лесах Нижнего мира. Это прожорливое чудовище будет
искать в качестве еды практически любое теплокровное животное, включая людей. Сначала он набрасывается на свою жертву и вцепляется в нее, высасывая досуха.
Насытившись, он начнет летать и использовать кровь своей жертвы против нее самой, выплевывая ее обратно в цель.
<NEWLINE>
Если убить багрового комара, надувшегося кровью, из него может выпасть мешочек с кровью. Он также может выронить свой хоботок. Оба
этих материала можно использовать для создания уникального оружия - Распылителя крови, которое использует комариную слюну для нанесения урона по площади.
В качестве боеприпасов для Распылителя крови используются мешочки с кровью.
<NEWLINE>
Если садовая муха из верхнего мира окажется в нижнем мире, она может превратиться в Багрового Комара.
